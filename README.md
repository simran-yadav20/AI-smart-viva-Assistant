# AI Smart Viva Assistant

An AI-powered viva assistant designed to simulate interview scenarios by generating questions, evaluating responses, and providing feedback using NLP and speech recognition.

## 🚀 Features
- Speech-to-text using Whisper
- AI-based question generation
- Answer evaluation using LLM
- Real-time feedback system
- Interactive interview simulation

## 🛠️ Tech Stack
- Python
- Whisper (Speech Recognition)
- Mistral / LLM
- NLP
- FastAPI / Streamlit (if used)

## 💡 How It Works
1. User speaks answer
2. Audio converted to text
3. AI generates next question
4. System evaluates response
5. Feedback is provided
